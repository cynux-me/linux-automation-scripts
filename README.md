# Linux Automation & Security Toolkit 🛡️

A collection of custom Bash scripts and Linux automation tools designed to streamline system administration, monitor server health, and perform essential security auditing tasks.

## 🚀 Features
* **System Monitoring:** Tracks server resource utilization, including RAM, CPU usage, and disk space in real-time.
* **Security Audit:** Inspects the system for potential vulnerabilities, misconfigurations, or unauthorized activities.
* **Automation:** Automates repetitive sysadmin tasks and cron job management.

## 🛠️ Technology Stack
* **Scripting Language:** Bash (Shell Scripting)
* **Environment:** Linux / Ubuntu OS

## 💻 How to Use
To run the script on your local machine or server, execute the following commands in your Linux terminal:

```bash
# Grant execution permission to the script
chmod +x linux-automation-scripts.sh

# Run the script
./linux-automation-scripts.sh
